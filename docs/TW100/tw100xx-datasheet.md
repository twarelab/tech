# TW100xx Datasheet

이 문서는 TW100xx 시리즈 제품의 하드웨어 정보를 제공합니다.

## Introduction
### Key Features
- 4 Port Serial
- Support DHCP IP Acquisition
- Support DNS Query
- Support NTP Time Query
- TCP/UDP Data Communication
- Support Ethernet Data Packing Options
- Support Up to 3Mbps UART Baud Rate
